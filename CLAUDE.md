# Globe Demo

Interactive demo for @aeryflux/globe package.

**Live**: [aeryflux.github.io/globe-demo](https://aeryflux.github.io/globe-demo/)

## Quick Start

```bash
npm install        # Install deps
npm run dev        # Dev server (port 5173)
npm run build      # Production build
npm run preview    # Preview build
```

## Port

**5173** (Vite default)

## Structure

```
globe-demo/
├── src/
│   ├── App.tsx       # Demo app
│   └── main.tsx      # Entry point
├── public/
└── vite.config.ts
```

## Tech Stack

- React 19 + Vite 7
- @aeryflux/globe
- Three.js
- TypeScript

## Features

- Interactive globe demonstration
- Theme switching (dark, green, white)
- Country/city click events
- Rotation controls

GitHub Pages (base path: `/globe-demo/`)
